# UI Components Library (Storybook Demo)

A collection of simple and reusable UI components demonstrated using **Storybook**.  
This project includes three main components:

- **SidebarMenu** — a slide-in sidebar menu with nested submenus
- **Input** — a versatile input with password toggle and clear button
- **Toast** — a notification component with success / error / info types

## Setup & Installation

### Clone repo

git clone https://github.com/Marakasss/develops_today_test_task.git
cd develops_today_test_task

### 2.Install dependencies

npm install

### 3. Run Storybook

npm run storybook

Storybook will be available at: http://localhost:6006/
